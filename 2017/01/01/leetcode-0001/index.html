0001. Two Sum

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

思路

  1. 暴力 时间复杂度 O(n2) 空间复杂度 O(1)
  2. 哈希 时间复杂度 O(n) 空间复杂度 O(n)）
  3. 排序再查找 时间复杂度 O(nlogn) 此题返回下标不可行，相似题可行

HashMap

我们考虑使用HashMap，拿空间换时间。
如果没有在哈希表中就插入，有就返回结果。

  • 执行用时: 0 ms
  • 内存消耗: 2.3 MB

需要注意，返回的是索引，所以插入哈希表时key应该是nums中的值，value才是index。

问题与优化思路

问题:
对Rust语法不算熟悉，需要查文档才能知道HashMap的用法。做算法题，开始得画图来理清自己的思路，确定需求和数据边界，这题中返回下标是首先要确定的需求。

优化思路:
X Sum的解题思路主要是把它转化为(X-1)Sum,最后到2Sum得到最后的答案
这其中的转换就是，选出一个元素a，然后就变成target为 -a 的（X-1)Sum 问题
